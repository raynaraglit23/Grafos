# EP1 - Algoritmo de Dijkstra (Grafos)

Este programa implementa o **Algoritmo de Dijkstra** em C para calcular e exibir o caminho de custo mínimo entre dois vértices em um grafo direcionado (digrafo). O grafo.txt é o do exemplo do professor.

## ⚙️ Como compilar o código

Para compilar o programa, abra o terminal no diretório onde o arquivo `ep1.c` está localizado e execute o seguinte comando:
```bash
gcc ep1.c -o ep1 -Wall
```
## ⚙️ Como rodar o código

Para compilar o programa, execute o seguinte comando no mac:
```bash
 ./ep1 grafo.txt
```
Para compilar o programa, execute o seguinte comando no Windowns:
```bash
 ep1.exe grafo.txt
