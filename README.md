# EP1 - Algoritmo de Dijkstra (Grafos)

Este programa implementa o **Algoritmo de Dijkstra** em C para calcular e exibir o caminho de custo mínimo entre dois vértices em um grafo direcionado (dígrafo).

## ⚙️ Como compilar o código

Para compilar o programa, abra o terminal no diretório onde o arquivo `ep1.c` está localizado e execute o seguinte comando:
```bash
gcc ep1.c -o ep1 -Wall
